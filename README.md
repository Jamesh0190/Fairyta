# FairyTales.tv - Magical Interactive Adventures!

A magical web application where children can create their own AI-powered interactive stories with fairytale characters and adventures.

## Features
- 🧚‍♀️ AI-generated stories
- 🎮 Interactive choice-based storytelling
- ✨ Beautiful animations and visual effects
- 👶 Child-friendly interface
- 📱 Fully responsive design

## Technologies Used
- HTML5
- CSS3 with custom animations
- Vanilla JavaScript
- LocalStorage for data persistence
- Lucide Icons
